METHOD

Thanks to the experience gained in over 5 years developing projects for international web agencies, I understand the importance of having organized, well documented and optimized code for excellent results.

I code always keeping in mind the motto of "... whatever you want them to do to you, do to others", which means that other programmers probably work over what I've coded, For that reason I work as organized as possible, resulting in a clear and easy to read code, so the teamwork will not become a nightmare.

To achieve this goal I use CSS preprocessors like Sass, Less or Stylus and apply variables, mixins and nested structures to speed up development in order to facilitate changes in the future.

ALWAYS use version control with Git and Github. As a personal rule, I daily study the latest standards and development projects to work with the latest standards. When a project requires it, I make customized developments for older browsers like IE < 9, Firefox < 33.

Finally I deliver each project with images, HTML, Javascript and CSS optimized and minified, separating units for each of the different development environments such as staging and production. This optimization will significantly help the SEO of your project decreasing load times and reduce the weight of the project.

Quality management is indispensable in professional development, so I perform tests on major browsers, mobile devices simulators and real devices testing to make sure each project is perfectly done, without bugs to achive a uniform designed project with the best quality.

Communication is essential to ensure the success of any project, for that reason I'll be in continuos communication with you and the team members designated for the project.
